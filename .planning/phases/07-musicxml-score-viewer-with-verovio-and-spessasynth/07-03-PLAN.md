**ScoreViewerControls.tsx** — Bottom playback bar (per UI-SPEC):
- `'use client'` directive
- Props: `{ isPlaying: boolean, currentTime: number, duration: number, isLoading: boolean, loadProgress: number, error: string | null, onPlay: () => void, onPause: () => void, onSeek: (time: number) => void }`
- Layout: horizontal flex with items-center gap-3, bg-card border-t p-3
- Play/Pause toggle button: Button size="icon" variant="default", 44px touch target (h-11 w-11)
  - Shows `Play` icon when paused, `Pause` icon when playing
  - Disabled when `isLoading` is true (with cursor-not-allowed, reduced opacity)
  - Tooltip: "Reproducir" / "Pausar"
- Seek bar: shadcn `Slider` component, min=0, max=duration, value=[currentTime], step=0.1
  - `onValueChange` calls `onSeek(value[0])`
  - Track color: `hsl(var(--border))` unfilled, `hsl(var(--primary))` filled
  - Flex-1 to fill available space
- Time display: `<span className="text-xs text-muted-foreground tabular-nums whitespace-nowrap">{fmt(currentTime)} / {fmt(duration)}</span>`
- When `isLoading && loadProgress > 0`: show Progress component below controls with text "Descargando sonidos... {loadProgress}%"
- When `error` is truthy: show muted banner "Audio no disponible. La partitura se puede ver pero no reproducir."
- `fmt()` function: `(s: number) => { if (!s || isNaN(s)) return '0:00'; return Math.floor(s/60) + ':' + Math.floor(s%60).toString().padStart(2,'0'); }`

**ScoreViewer.tsx** — Main component (per D-04, D-09, D-10, D-11):
- `'use client'` directive
- Props interface:
  ```typescript
  interface ScoreViewerProps {
    musicxmlFileId: string;
    midiFileId?: string;
    className?: string;
  }
  ```
- Export default function `ScoreViewer`
- Uses all three hooks: `useVerovio()`, `useSpessaSynth()`, `useScoreCursor({...})`
- State: `currentPage: number` (1-based), `scale: number` (default 40), `scoreLoaded: boolean`, `svgContent: string` (current page SVG)
- Refs: `containerRef = useRef<HTMLDivElement>(null)`, `scoreContainerRef = useRef<HTMLDivElement>(null)`
- On mount + toolkit ready: fetch MusicXML from `/api/hymns/score/${musicxmlFileId}`, parse as text, call `loadScore(xmlText)`
- After score loaded: call `renderPage(1)` to get first SVG, set svgContent. Start SoundFont preload via `loadSoundFont()`.
- After SoundFont loaded: call `getMidi()` to get base64 MIDI, call `loadMidi(midiBase64)` (per Research: use Verovio-generated MIDI, NOT original)
- Playback flow: play button calls `play()`, pause calls `pause()`, seek calls `seek(time)`
- Cursor sync: `useScoreCursor` with `onPageChange` callback that calls `renderPage(newPage)`, updates svgContent, and scrolls container to top with `scrollIntoView({ behavior: 'smooth', block: 'start' })`
- Zoom: handlers call `setScale(newScale, containerWidth)` then re-render current page. Scale bounds: min 20, max 80, step 10.
- Responsive scale: use `useEffect` with ResizeObserver on containerRef to get width, set initial scale based on breakpoint per UI-SPEC:
  - < 640px: scale 30
  - 640-1024px: scale 35
  - > 1024px: scale 40
- Keyboard shortcuts (via onKeyDown on container div with tabIndex={0}):
  - Space: toggle play/pause (preventDefault to avoid scroll)
  - ArrowLeft: seek back 5 seconds
  - ArrowRight: seek forward 5 seconds
  - `+` or `=`: zoom in
  - `-`: zoom out
  - `0`: reset zoom
- SVG rendering: use `dangerouslySetInnerHTML={{ __html: svgContent }}` on the score container div
- Inject CSS style tag for note highlighting (per UI-SPEC interaction contract):
  ```html
  <style>{`.score-container g.note.playing { fill: hsl(var(--primary)); transition: fill 0.05s ease; } .score-container g.note.playing rect { fill: hsl(var(--primary) / 0.15); }`}</style>
  ```
- Loading states per UI-SPEC:
  - WASM loading: Skeleton placeholder (h-[400px] w-full) with centered Loader2 animate-spin + "Cargando motor de partitura..." text, bg-muted/50
  - MusicXML loading: Same skeleton with "Cargando partitura..." text
  - SoundFont loading: Score visible, progress shown in ScoreViewerControls
  - Error WASM: red alert banner "No se pudo cargar el visor de partituras. Intenta recargar la pagina."
  - Error MusicXML: empty state with ScrollText icon + "No se pudo cargar la partitura. Intenta recargar la pagina."
  - Error SoundFont: score renders, controls disabled, muted banner via ScoreViewerControls error prop
- Layout structure:
  ```
  <div className={cn("flex flex-col border rounded-lg overflow-hidden", className)}>
    <ScoreToolbar ... />
    <Separator />
    <div ref={scoreContainerRef} className="score-container relative overflow-y-auto" style={{ minHeight: '400px', maxHeight: '70vh' }}>
      <style>...</style>
      <div ref={containerRef} dangerouslySetInnerHTML={{ __html: svgContent }} />
    </div>
    <ScoreViewerControls ... />
  </div>
  ```
