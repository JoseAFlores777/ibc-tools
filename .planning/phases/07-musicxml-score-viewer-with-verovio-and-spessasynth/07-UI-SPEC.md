## Component Inventory

### ScoreViewer (new)

The main reusable component. `'use client'`, dynamically imported with `ssr: false`.

**Layout (desktop):**
```
+-------------------------------------------------------+
| [Toolbar: Zoom -/+ | Page X/N | Tempo display]        |
+-------------------------------------------------------+
|                                                         |
|                   SVG Score Area                         |
|              (auto-width, scrollable Y)                 |
|                                                         |
+-------------------------------------------------------+
| [Play/Pause] [Seek Bar =========o---------] [0:45/2:30]|
+-------------------------------------------------------+
```

**Layout (mobile):**
```
+-----------------------------------+
| [Zoom -/+] [Pg X/N] [Tempo]      |
+-----------------------------------+
|                                   |
|         SVG Score Area            |
|     (full width, scroll Y)       |
|                                   |
+-----------------------------------+
| [Play] [====o------] [0:45/2:30] |
+-----------------------------------+
```

**Props:**
- `musicxmlFileId: string` -- Directus file UUID for MusicXML
- `midiFileId?: string` -- Optional original MIDI file UUID (for non-score playback contexts)
- `className?: string`

**States:**

| State | Visual |
|-------|--------|
| Loading WASM | Skeleton placeholder (full width, 400px height) with centered Loader2 spinner and "Cargando motor de partitura..." text |
| Loading MusicXML | Same skeleton with "Cargando partitura..." text |
| Loading SoundFont | Score renders immediately (SVG visible). Overlay on playback controls: Progress bar with "Descargando sonidos... X%" |
| Ready (idle) | SVG score displayed, Play button enabled, seek bar at 0:00 |
| Playing | Play icon swaps to Pause icon. Seek bar fills. Notes highlight with cursor. Auto-scroll follows. |
| Paused | Pause icon swaps back to Play. Cursor stays on last note. Seek bar holds position. |
| Ended | Resets to start. Play button shows Play icon. Seek at 0:00. |
| Error (WASM) | Red alert banner: "No se pudo cargar el visor de partituras. Intenta recargar la pagina." |
| Error (SoundFont) | Score still renders (SVG visible). Playback controls disabled. Muted banner: "Audio no disponible. La partitura se puede ver pero no reproducir." |
| Error (MusicXML) | Empty state: ScrollText icon (muted) + "No se pudo cargar la partitura" |

### ScoreViewerControls (new)

Bottom control bar for playback.

| Control | Icon | Behavior |
|---------|------|----------|
| Play/Pause | `Play` / `Pause` (Lucide) | Toggle playback. 44px touch target. |
| Seek bar | shadcn Slider | Horizontal slider. Thumb shows current position. Drag to seek. |
| Time display | Text label | `currentTime / duration` in `M:SS` format. Font: monospace variant via `tabular-nums` class. |
| Tempo display | Badge | Shows current BPM (from MIDI). Read-only for v1. |

### ScoreToolbar (new)

Top toolbar for non-playback controls.

| Control | Element | Behavior |
|---------|---------|----------|
| Zoom out | Button (icon-only, `ZoomOut`) | Decrease Verovio scale by 10. Min scale 20. |
| Zoom in | Button (icon-only, `ZoomIn`) | Increase Verovio scale by 10. Max scale 80. |
| Zoom reset | Button (text, "100%") | Reset to default scale (40). |
| Page indicator | Text label | "Pag. X / N" -- updates during playback auto-scroll. |

### MidiPlayer (new, replaces MidiTrackPlayer)

Drop-in replacement for MidiTrackPlayer. Same visual appearance (compact card row with play button, progress bar, download). Uses SpessaSynth internally instead of Tone.js.

**Props interface must match MidiTrackPlayer:**
- `field: string`
- `fileInfo: AudioFileInfo`
- `label: string`
- `colorClass: string`
- `hymnName?: string`

Visual output is identical to current MidiTrackPlayer -- no visual changes. Only the internal engine changes.

### Partituras Tab (modification to HymnDetailView)

The existing placeholder in `HymnDetailView` (lines 659-665) is replaced with:

**When hymn has MusicXML:**
- Renders `<ScoreViewer musicxmlFileId={...} />` filling the tab content area
- Min height: 400px
- Max height: 70vh (with internal scroll for multi-page scores)

**When hymn has no MusicXML:**
- Keep existing placeholder: ScrollText icon + "Partitura" + "Proximamente"
- No changes to this state

## Interaction Contracts

### SVG Note Highlighting

When a note is playing, its SVG `<g>` element receives the CSS class `playing`. The style is applied via a `<style>` tag injected into the score container (not inline styles on SVG elements):

```css
.score-container g.note.playing {
  fill: hsl(var(--primary));
  transition: fill 0.05s ease;
}
.score-container g.note.playing rect {
  fill: hsl(var(--primary) / 0.15);
}
```

The `playing` class is added/removed via direct DOM manipulation in the `requestAnimationFrame` sync loop, not via React state (performance requirement).

### Auto-Scroll

During playback, when `getElementsAtTime()` returns a page number different from the currently displayed page:
1. Render the new page SVG via `renderToSVG(pageNumber)`
2. Swap the container innerHTML
3. Smooth scroll the container to top: `scrollIntoView({ behavior: 'smooth', block: 'start' })`

### Seek Interaction

- Dragging the slider calls `sequencer.currentTime = newTime` on SpessaSynth
- The cursor position updates immediately (no waiting for next RAF tick)
- The time display updates in real time during drag

### Zoom Interaction

- Zoom changes call `tk.setOptions({ scale: newScale })` then `tk.redoLayout()`
- All pages re-render. Current page is preserved.
- Container scroll position adjusts to keep the same vertical content visible

### Responsive Behavior

| Breakpoint | Behavior |
|------------|----------|
| < 640px (mobile) | Score renders at scale 30. Toolbar items stack if needed. Controls use compact layout. |
| 640-1024px (tablet) | Score renders at scale 35. Standard layout. |
| > 1024px (desktop) | Score renders at scale 40. Full toolbar. Two-column layout in HymnDetailView. |

### Keyboard Shortcuts (within ScoreViewer focus)

| Key | Action |
|-----|--------|
| Space | Toggle play/pause |
| Left Arrow | Seek back 5 seconds |
| Right Arrow | Seek forward 5 seconds |
| + / = | Zoom in |
| - | Zoom out |
| 0 | Reset zoom |
