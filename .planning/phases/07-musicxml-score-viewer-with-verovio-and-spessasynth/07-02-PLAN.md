  ```typescript
  });
  ```
- `loadScore(musicXmlString: string)`: calls `tk.loadData(musicXmlString)` then `tk.renderToMIDI()` (CRITICAL: must call renderToMIDI after loadData to build timing map â€” Research Pitfall 4)
- `renderPage(pageNum: number)`: returns `tk.renderToSVG(pageNum)` string

**useSpessaSynth.ts** â€” SpessaSynth synthesizer and sequencer lifecycle:
- Export `useSpessaSynth()` hook returning `{ synth, sequencer, isLoading, loadProgress, error, loadSoundFont, loadMidi, play, pause, seek, duration, currentTime, isPlaying, onEnded }`
- State: `synth: WorkletSynthesizer | null`, `sequencer: Sequencer | null`, `isLoading: boolean`, `loadProgress: number` (0-100), `error: string | null`, `isPlaying: boolean`, `duration: number`, `currentTime: number`
- `loadSoundFont()`: 
  1. Check IndexedDB cache first (DB name: `'ibc-audio-cache'`, store: `'files'`, key: `'ibc-soundfont-v1'`)
  2. If cached, use cached ArrayBuffer
  3. If not cached, fetch from `/api/hymns/soundfont` with progress tracking via `response.body.getReader()` â€” update `loadProgress` state as chunks arrive
  4. Cache the downloaded buffer in IndexedDB for next time
  5. Create AudioContext, add worklet module: `await ctx.audioWorklet.addModule('/spessasynth_processor.min.js')`
  6. Create synth: `new WorkletSynthesizer(ctx)`
  7. Load soundbank: `await synth.soundBankManager.addSoundBank(buffer, 'main')`
  8. `await synth.isReady`
- `loadMidi(midiBase64: string)`:
  1. Decode base64 to Uint8Array: `Uint8Array.from(atob(midiBase64), c => c.charCodeAt(0))`
  2. Create sequencer if not exists: `new Sequencer(synth)`
  3. Load MIDI: `seq.loadNewSongList([midiBytes.buffer])`
  4. Register event handlers: `songEnded` resets isPlaying, `timeChange` for seek updates
  5. Read duration from sequencer
- `play()`: calls `sequencer.play()`, sets isPlaying true, starts RAF loop for currentTime updates
- `pause()`: calls `sequencer.pause()`, sets isPlaying false
- `seek(timeSeconds: number)`: calls `sequencer.currentTime = timeSeconds`
- `onEnded`: callback ref for songEnded event
- Cleanup on unmount: pause sequencer, close AudioContext via `ctx.close()`

**useScoreCursor.ts** â€” RAF-based cursor synchronization:
- Export `useScoreCursor(options)` hook where options: `{ toolkit: VerovioToolkit | null, sequencer: Sequencer | null, isPlaying: boolean, containerRef: RefObject<HTMLDivElement>, onPageChange: (page: number) => void }`
- Returns `{ currentPage: number }`
- When `isPlaying` is true, start a `requestAnimationFrame` loop:
  1. Read `sequencer.currentHighResolutionTime * 1000` (convert to ms)
  2. Remove `.playing` class from all `g.note.playing` elements â€” MUST use `containerRef.current.querySelectorAll('g.note.playing')` (scoped to container, NOT `document.querySelectorAll`)
  3. Call `toolkit.getElementsAtTime(timeMs)`
  4. If `result.page > 0` and differs from current page, call `onPageChange(result.page)` (per D-11 auto-scroll)
  5. For each noteId in `result.notes`, find element by ID within containerRef using `containerRef.current.querySelector('#' + noteId)` and add `.playing` class
  6. Continue loop with next `requestAnimationFrame` if still playing
- When `isPlaying` becomes false, cancel the RAF loop and clear all `.playing` classes (scoped to containerRef)
- Use `useRef` for the RAF id to allow cancellation
- Direct DOM manipulation for performance (not React state â€” per UI-SPEC interaction contract)
