    **1. Create `app/components/pdf-components/pdf-pages/HymnPageBooklet.tsx`:**

    This is a VIEW component (not a Page) that renders a single hymn's content sized for a half-letter booklet page (396x612pt). It is rendered INSIDE a BookletSheet slot.

    Props interface:
    ```typescript
    export interface HymnPageBookletProps {
      hymn: HymnForPdf;
      verses: ParsedVerse[];
      fontPreset: FontPreset;       // from pdf-tokens
      includeBibleRef: boolean;     // D-09
      style: 'decorated' | 'plain'; // determines visual treatment
    }
    ```

    Import font preset from `FONT_PRESETS_BOOKLET` in pdf-tokens.ts. Use `props.fontPreset` to look up family and scale.

    Layout structure (all within a View, NOT a Page — BookletSheet provides the Page):
    - **Header area** (compact): Hymn number + title in a single row. If `style === 'decorated'`, use a compact colored bar (background `COLORS.headerBg`, text white, height ~40pt). If `style === 'plain'`, use plain text header (number + title, no background).
    - **Bible reference** (conditional): If `includeBibleRef && hymn.bible_text`, render bible text in italic and reference in small text. If `!includeBibleRef`, skip entirely (do not render the View).
    - **Verses area** (flex: 1): Map over `verses` array. Each verse has a label (bold, using `scale.body` size) and lines (regular, using `scale.body` size). Chorus lines in italic.
    - **Hymnal name** (footer-like): Small text at bottom showing `hymn.hymnal?.name` if present.

    For decorated style in booklet: header branding elements (church name text if any) always use Adamina font regardless of fontPreset. Lyric body uses the preset's family. This is per the UI-SPEC note about decorated style.

    Dimensions: The component should set `width: BOOKLET_PAGE_WIDTH - 2 * BOOKLET_MARGIN` and `height: BOOKLET_SHEET_HEIGHT - 2 * BOOKLET_MARGIN` as its outer View dimensions, with `padding: BOOKLET_MARGIN`.

    Actually, simplify: the outer View should have `width: BOOKLET_PAGE_WIDTH`, `height: BOOKLET_SHEET_HEIGHT`, `padding: BOOKLET_MARGIN`. This means the BookletSheet positions two of these side by side.

    Use named export: `export function HymnPageBooklet(props: HymnPageBookletProps)`.
