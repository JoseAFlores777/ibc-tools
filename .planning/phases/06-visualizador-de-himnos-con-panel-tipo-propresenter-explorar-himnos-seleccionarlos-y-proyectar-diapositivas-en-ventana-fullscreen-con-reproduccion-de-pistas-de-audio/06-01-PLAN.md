    **app/visualizador/hooks/useVisualizador.ts** — Implements the `visualizadorReducer` function and `useVisualizador` hook using `useReducer`. Import types from `../lib/types`. Export both the reducer (for testing) and the hook. Initial state: empty playlist, activeHymnIndex -1, activeSlideIndex 0, projectionMode 'slide', projectionOpen false, theme from DEFAULT_THEME, audio { hymnId: null, trackField: null, playing: false }.

    Key reducer logic:
    - `ADD_HYMN`: Build slides from hymn data using `buildSlideGroups(parseHymnHtmlClient(hymn.letter_hymn ?? ''), hymn)`, push to playlist. If playlist was empty, set activeHymnIndex=0.
    - `REMOVE_HYMN`: splice playlist, clamp activeHymnIndex. If playlist becomes empty, set activeHymnIndex=-1.
    - `REORDER_PLAYLIST`: use arrayMove pattern (splice out, splice in). Adjust activeHymnIndex if it was moved.
    - `NEXT_SLIDE`: if activeSlideIndex < slides.length - 1, increment. Else if activeHymnIndex < playlist.length - 1, set activeHymnIndex+1 and activeSlideIndex=0 (per D-07 auto-advance).
    - `PREV_SLIDE`: if activeSlideIndex > 0, decrement. Else if activeHymnIndex > 0, set activeHymnIndex-1 and activeSlideIndex to last slide of previous hymn.
    - `SET_ACTIVE_HYMN`: set activeHymnIndex, reset activeSlideIndex=0. If audio.hymnId !== new hymn id, stop audio (D-20).
    - `FONT_SIZE_UP/DOWN`: adjust fontSizeOffset by +/-4, clamped to [-20, +20].
    - `SET_AUDIO_TRACK`: set audio.hymnId and trackField. If hymnId changed from previous, set playing=false.

    **app/visualizador/hooks/useBroadcastChannel.ts** — Hook that creates a BroadcastChannel on mount and closes on unmount. Returns `{ send(msg: ProjectionMessage): void }`. Accepts optional `onMessage` callback. Uses `useRef` for channel and `useEffect` for lifecycle. Channel name from `CHANNEL_NAME` constant.

    **app/visualizador/hooks/useAutoFontSize.ts** — Hook that accepts `{ text: string; containerWidth: number; containerHeight: number; fontFamily: string; sizeOffset: number }` and returns a computed font size number. Uses `useMemo` internally. Binary search between 16px and 120px using a hidden `<canvas>` created once (useRef). For each candidate size, measure all lines with `ctx.measureText()`, compute total height with `lineHeight = size * 1.4`, check both max line width <= containerWidth - 160 (80px padding each side) and total height <= containerHeight - 160. Return `Math.max(16, lo + sizeOffset)`.

    **app/visualizador/hooks/useKeyboardShortcuts.ts** — Hook that accepts `{ dispatch: React.Dispatch<VisualizadorAction>; togglePlayPause: () => void; projectionOpen: boolean }`. Registers a global `keydown` listener via `useEffect`. Handles: ArrowRight -> NEXT_SLIDE, ArrowLeft -> PREV_SLIDE, Space -> NEXT_SLIDE, B -> SET_PROJECTION_MODE 'black' toggle, L -> SET_PROJECTION_MODE 'logo' toggle, C -> SET_PROJECTION_MODE 'clear' toggle, P -> togglePlayPause, Ctrl+= (Plus) -> FONT_SIZE_UP, Ctrl+- (Minus) -> FONT_SIZE_DOWN. Call `e.preventDefault()` on all handled keys. Do NOT handle if `e.target` is an `<input>` or `<textarea>` (user may be typing in search).
